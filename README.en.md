# Linkboard

A cross-device clipboard sync and history manager supporting LAN, Bluetooth, and WebRTC direct transfer. Works out of the box with no cloud dependency; data stays on your devices and is transferred/stored locally.

[中文文档](README.md)

## Features
- Direct transfer over LAN (TCP), Bluetooth (BLE), and WebRTC
- Uses TCP on the same LAN; falls back to other modes when direct connection is not possible
- Cross-platform: Android, iOS, Windows, macOS
- Sync text and images (auto-detect PNG/JPEG)
- History management with configurable retention days and one-tap cleanup
- App keyboard extension: paste from history without leaving the current app
- Fully local: no cloud services, no data collection
